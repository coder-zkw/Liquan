TestStorm200 (omnipotent)
product manual:

1. A new generation of portable all-round test frame, modular design, providing 2 slots, supporting any combination of test modules from 10M to 10G.

2. A portable platform that provides 2-7 layers of Ethernet testing to meet the professional testing requirements of communication protocols in research and development, experimentation, network monitoring, quality control, and production.

3. It can realize 2-7 layer mixed flow test and protocol simulation, and it can be used together with V series test module.

4, support IPv4 and IPv6 protocol stack, a single port can simulate thousands of interfaces, simulate thousands of real user connections, provide test, simulation and conformance test solutions for 2-7 layers of complex protocols.

Key features:

Support 2-7 layer protocol simulation to meet high-end R&D testing needs

Support network sharing remote operations to improve efficiency

Support multi-chassis cascading, all port synchronization test

Graphical operation software platform

Automate configuration of TCL scripts

Standard RFC2544/2889/3918 test kit

X series test module:
The X-Series test modules are based on the full range of high-performance professional test modules from the TestStorm family of network testers. Support all the features of the TestStorm series of network testers to provide customers with the best price/performance ratio.
The X-series test modules are equipped with professional multi-core processors of 36 cores and above, with powerful data calculation and processing capabilities, and protocol simulation performance specifications have reached the industry-leading level.
The X series test modules are divided according to different port speeds. The X6000 series test modules support 10/100/1000M speed RJ45 interfaces or 100/1000M speed SFP interfaces, and the X8000 series test modules support 10G speed SFP+ interfaces.
