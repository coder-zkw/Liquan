BigTao210 (Production)
product manual:

1. A new generation of portable production test frame, modular design, providing 2 slots, supporting any combination of test modules from 10M to 100G.

2. Add portable handles on the side, the maximum running noise is only 65 decibels, and the maximum power consumption of the whole machine is only 150W.

3. Provide TeleTIN, a large-scale automated test software for production lines, for use with P-series test modules.

4. Provide 2-3 layer Ethernet traffic test and protocol simulation to meet the test requirements of large-scale production, quality control, and general R&D.

5. It can be customized with the factory MES system and API secondary development to improve the quality and efficiency of the communication factory.

6. If you want to use multiple slots at the same time, please purchase the host BigTao6100 with 6 slots.

Key features:

Support independent host use, convenient and flexible

Support network sharing for multi-person use, improve efficiency

Support multi-chassis cascading, all port synchronization test

Graphical operation software platform

Automate configuration of TCL scripts

API for secondary development

P series test module:
The P series test module is a production test module based on the BigTao tester platform. It can realize 2~3 layer flow test for network equipment and network system, and meet the test requirements in the process of production and R&D quality control.
Designed for large-scale production testing, the P-Series modules are equipped with dedicated production test software TeleATT. The test process is comprehensive, efficient and automated, providing a cost-effective test solution for production line testing.
